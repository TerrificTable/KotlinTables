# Tables
Simple, really simple, "library" to create tables in kotlin <br>
Includes helper class to read (and kinda parse) csv files

<br>

### Example
(src: examples/Simple.kt) 
```
firstname  lastname

Peter      Griffin   
Lois       Griffin   
Glenn      Quagmire
```
